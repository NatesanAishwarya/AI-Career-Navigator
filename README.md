# AI Career Navigator

AI Career Navigator is a backend project built using FastAPI, PostgreSQL, SQLAlchemy, bcrypt, and JWT Authentication.

## Features

* User Registration
* User Login
* Duplicate Email Validation
* Password Hashing (bcrypt)
* JWT Token Authentication
* PostgreSQL Database Integration

## Tech Stack

* FastAPI
* PostgreSQL
* SQLAlchemy
* bcrypt
* python-jose (JWT)

## API Endpoints

* POST /register
* POST /login
* GET /users

## Status

Currently under development.
